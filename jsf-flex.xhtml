JSF Flex project
http://jsf-flex.googlecode.com is the namespace URL for JSF Flex project, which aim is to provide capability in creating standard Flex components as normal JSF components [so technically this file should exist within that url, perhaps in the future change the namespace URL to the location of this file] This will allow the same usage of JSF MVC architecture by having :
  • XHTML and JSP tags generating MXML, SWC, SWF, and etctera
  • Managed beans of JSF binding data to Flex components by either
    • multi asynchronous data request for large data set [i.e. DataGrid]
    • linking of values during form submit by using JSON+Javascript and ActionScript [i.e. RichTextEditor]

Component Value Mapper SNAPSHOT of &release;
This file provides fields that contain pertinent values for mapping Flex components to managed beans on the server side. Meaning during the form submit or in any other process where Flex component's value need to be looked up, the JSF Flex ActionScript code will
  • reflect on the specific flex component
  • get the pertinent value by searching within componentValueMapper.xml for the reflected flex component
  • and return the value in JSON object format
